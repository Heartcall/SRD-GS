## Interpretation

- The Shiny Blender Synthetic `ball` dataset has a usable test split when `eval=True`.
- `points3d.ply` must not be silently promoted to accepted GT geometry. In this checkout, the Ref-GS Blender reader can generate and store a random point cloud at that path.
- SRD branch maps remain fallback metadata rather than true rasterized, differentiable branch-gate/specular-weight/transport buffers.
- Broad paper-scale runs should remain blocked until accepted GT geometry and true SRD branch-map rasterization are resolved.
